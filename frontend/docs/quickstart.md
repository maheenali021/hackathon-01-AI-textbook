# Quick Start Guide

Get started with Physical AI & Humanoid Robotics in minutes.

## Prerequisites

- Node.js 18 or higher
- npm or yarn package manager
- Git for version control

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/maheenali021/hackathon-01-AI-textbook.git
   cd hackathon-01-AI-textbook
   ```

2. **Navigate to the frontend directory**
   ```bash
   cd frontend
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

## Running the Development Server

1. **Start the development server**
   ```bash
   npm start
   ```

2. **Open your browser**
   Visit [http://localhost:3000](http://localhost:3000) to view the documentation.
